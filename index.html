Audio samples from "Code-switching speech synthesis for Mandarin-English using FastSpeech2: A unified IPA-based approach"

Author: Wang Yinqiu

1. Modeling the phonemes of Mandarin and English separately:

To model the phonemes of Mandarin and English separately, I conducted three sets of experiments:

  • (1) Group A: Pre-training the model using a code-switched Mandarin-English dataset for ASR tasks and then fine-tuning with 500 high-quality code-switched Mandarin-English audios;
  • (2) Group B: Pre-training the model with a dataset for TTS tasks in Mandarin, followed by fine-tuning using 500 high-quality code-switched Mandarin-English audios;
  • (3) Group C: Pre-training the model with a dataset for TTS tasks in Mandarin, followed by fine-tuning with a TTS English dataset.

2. Unifying the input formats for both languages as phonological features based on the IPA:

Group D: The input formats for both languages were unified as phonological features based on the IPA, fine-tuning the pre-trained model with only 500 high-quality mixed Mandarin-English sentences (the pre-trained model is sourced from the IMS-Toucan repository on GitHub).
